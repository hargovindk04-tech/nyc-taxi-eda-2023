# New York City Yellow Taxi Data Analysis

## Objective
This case study focuses on exploratory data analysis (EDA) utilizing a dataset of yellow taxi rides in New York City. The goal is to demonstrate why EDA is a critical step in the data science and machine learning process.

## Problem Statement
As an analyst for an upcoming taxi operation in NYC, the objective is to use 2023 taxi trip data to discover insights that can optimize taxi operations. The analysis aims to uncover data patterns that inform strategic decisions to improve service efficiency, maximize revenue, and enhance the passenger experience.

## Project Tasks
The analysis in 'EDA_NYC_Taxi_Analysis.ipynb' includes the following steps:
* Data Loading
* Data Cleaning
* Exploratory Analysis: Bivariate and Multivariate
* Creating Visualisations to Support the Analysis
* Deriving Insights and Stating Conclusions

## Dataset
* **Timeframe:** Data from 2023.
* **Format:** Parquet ('.parquet').
* **Source:** Collected and provided to the NYC Taxi and Limousine Commission (TLC) by technology providers. [TLC Trip Record Data](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page).

## Getting Started
1. Clone this repository.
2. Install the required dependencies using 'pip install -r requirements.txt'.
3. Download the 2023 Yellow Taxi trip records from the TLC website and place them in the appropriate data directory.
4. Run the Jupyter Notebook 'EDA_NYC_Taxi_Analysis.ipynb' to view the data loading, cleaning, and exploratory analysis.
